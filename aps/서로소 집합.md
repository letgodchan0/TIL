## 서로소 집합

> 서로소 집합(상호배타 집합)은 서로 중복 포함된 원소가 없는 집합들(교집합이 없는 집합)

- 표현 방법
  - 연결 리스트
  - 트리
- 상호배타 집합 연산
  - Make-Set(x)
  - Find-Set(x)
  - Union(x, y)

#### Make-Set(x)

> 유일한 멤버 x를 포함하는 새로운 집합을 생성하는 연산

```
Make-Set(x)
	p[x] <- x
```

#### Find-Set(x)

> x를 포함하는 집합을 찾는 연산

```
Find-Set(x)
	IF x == p[x]: RETURN x
	ELSE: RETURN Find-Set(p[x])
	
# 반복
Find-Set(x)
	while x != p[x]
		x = p[x]
	return x
```

#### Union(x, y)

> x와 y를 포함하는 두 집합을 통합하는 연산

```
Union(x, y)
	p[Find-Set(y)] <- Find-Set(x)
```
