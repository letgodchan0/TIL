# 목록

## 순서가 없는 목록 (unorderd list)

목록은 `*`나 `-`을 통해서 작성 가능하다. 

`tab`을 이용하면 하위 목록으로 전환 가능하다.

- 목록 1
- 목록2
  - 목록 2-1



## 순서가 있는 목록 (orderd list)

`1.` 으로 목록을 구성한다. 

1. 세수
2. 아침
3. 출근
